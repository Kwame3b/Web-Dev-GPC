What is the internet?

It is a network of networks that connects many computers and devices around the world.

How does the internet work?

The internet works by breaking data into small pieces called packets and sending them between devices using rules called protocols.

What is a domain name?

A domain name is the name you type in the browser to visit a website. Example: google.com

What is DNS?

DNS means Domain Name System. It changes domain names into IP addresses so your browser can find the website.

What is a browser?

A browser is a program used to open and view websites. Example: Chrome, Firefox.

What is a markup language?

A markup language is used to structure and format documents. Example: HTML.

What is frontend development?

Frontend development is building the parts of a website that users see and interact with.

What is HTTP?

HTTP is a system used for sending data between your browser and a website.

What is the web?

The web is a collection of websites and pages you can access through the internet.
